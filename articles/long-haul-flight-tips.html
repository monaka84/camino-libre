長距離フライトの過ごし方
エコノミーでも少し快適に過ごす工夫
【装備編②】

Monaka · 2026年4月 · 10分で読める
← 一覧に戻る

中南米への直行便は日本からなく、北米経由で乗り継ぐと片道20時間を超えることもざらにある。長距離フライトはどうしても疲れるが、機内に「何を持ち込むか」を少し工夫するだけで体への残り方が変わる、というのが何度かのフライトで掴んだ感覚だ。

ただしエコノミー席はとにかくスペースがない。優先順位は「小さく使える」「出し入れしやすい」「なくしても困りにくい」の3つ。この軸でいつも持ち込んでいるものを紹介する。

寝る——空気式ネックピロー

ネックピローは以前しっかりしたタイプを2つ買ったが、首は支えられるもののとにかく荷物になるのが苦手で、結局使わなくなった。

その点、空気式のTabineは使わないときに小さく畳める。低反発のような「がっちり支える感じ」ではないが、首を少し支える用途としては十分。荷物を増やしたくない人と相性がいい。

Tabine minimalU+ 空気式ネックピロー

旅寝工房 / 型番 minimalU+ / インフレータブル / 厚さ4cm・卵3つより軽い超薄型 / 空気ポンプ内蔵収納ケース / カラビナ付 / コットン調生地・防汚性

使わないときはかなり小さくできるのが魅力。空気量で首の支え具合を調整でき、低反発タイプのような「がっちり支える感じ」ではないが、機内やバスでの移動中に首を支える用途としては十分。収納ケースが空気ポンプを兼ねているので肺活量に頼らず膨らませられ、カラビナでバッグに外付けできるのも荷物を増やしたくない人と相性が良い。

光と音を遮る——アイマスクと耳栓

機内は思った以上に光と音の刺激が多い。隣の人のスクリーン光、エンジン音、カートの音と、寝つきたい時に限って邪魔が揃っている。

そこで毎回ポーチに入れているのがアイマスクと耳栓。どちらも軽くて小さく、いざ眠りたいときに「持ってきてよかった」と毎回思う。耳栓は左右がコードでつながっているタイプを選ぶのが大事で、これがないと寝ている間に外れて片方をなくす。アイマスクはヒモが細すぎないものの方がズレにくい。

ゴーウェル コード付き耳栓 GW-1502

Gowell / 型番 GW-1502-001 / シリコン製 / コード付き / 繰り返し使用可

シリコンの3段フランジで耳にフィットしやすい。何よりも左右をつなぐネックストラップが重要で、これがないと寝ている間に外れて片方なくす。コード付きを選ぶだけで、機内での「耳栓ロスト」がほぼなくなる。シリコン素材なので繰り返し洗って使えて、ポーチに入れっぱなしにできるのも便利。

CLAXIA SLEEPMASK 立体型アイマスク

CLAXIA / 型番 CLA-001 / 立体3D構造 / 完全遮光 / 東大医学部睡眠専門医 監修

アイマスクは正直どれでも構わないが、機内で配られる薄いタイプより自分で買ったものの方が確実に快適だった。立体型は目の周りに圧がかからず、まつ毛にも触れない。完全遮光で隣席のスクリーン光や窓側のシェード光を遮れる。

映画と音——有線イヤホンと2ピン変換アダプタ

機内エンタメは普段Bluetoothイヤホン派だが、長距離フライトに関しては有線に戻した。一度のフライトでBluetoothイヤホンを2回落としたのがきっかけだ。1回目は寝ている間に外れて後ろの席の人が拾ってくれ、2回目は座席の隙間に飛んでいきCAさんに「見つかってよかったですね」と笑われた。

それ以来、final E3000のような手頃な有線イヤホンを基本にしている。なくした時のダメージが小さく、気を張らなくていいのが楽だ。

final E3000 カナル型有線イヤホン

final / 型番 FI-E3DSS / 3.5mm有線 / カナル型 / イヤーピース5サイズ・イヤーフック・ポーチ同梱

値段の割に音はしっかりしていて、機内モニターの音源には十分。なくした時の心理的ダメージが小さく、Bluetoothを座席で2回落とした経験から、長距離フライトでは有線に戻った。イヤーピースが5サイズ付属するので耳に合わせやすい。

ワイヤレス派には Bluetoothトランスミッター

「ノイキャンのワイヤレスで映画を見たい」という人にはBluetoothトランスミッターという手もある。機内モニターのイヤホンジャックに挿せば、モニター音声を自分のワイヤレスイヤホンに飛ばせる。Twelve South AirFlyが定番で、2ピン端子にも対応。ただし寝落ちしやすい人はイヤホン紛失のリスクは残るので、自分の睡眠スタイル次第で。

仕事資料は iPad・スマホに入れておく

エコノミー席でノートパソコンを開くのは、前の人がリクライニングを倒した瞬間に詰む。出張で資料に目を通したい時は、PDFやスライドを事前にiPad・スマホへローカル同期しておくのが現実的。クラウド頼みだと機内Wi-Fiが不安定な時に困るので、出発前の5分でダウンロードを済ませておくとストレスが減る。

足元——機内スリッパより、最初から楽な靴

機内用にスリッパやサンダルを持ち込む人もいるが、個人的には脱いだ靴の置き場に困るのが気になる。とはいえサンダル一本だと現地で歩けない——中南米のように石畳や砂利の道を歩く場面が出ると、つま先や足の裏に負担がかかる。

そこで最近落ち着いたのが、かかとを踏める2way系のシューズだ。KEEN HOODMOC HSのような、つま先は覆われていてかかとを起こせばシューズ、踏めばスリップオンのように履けるタイプ。機内では踵を踏んで楽に履き、着いたら足を覆って歩ける。サンダルと靴を別々に持つ必要がない。

KEEN HOODMOC HS（フッドモック エイチエス）

KEEN / HOODコレクション / バンジーシューレース / 防水（トゥ・3/4） / コンバーチブル（踵踏み可） / ソール厚2cm

つま先と前足部はしっかり覆われていて、踵を起こせばシューズ、踏めばスリップオン／サンダルのように使えるコンバーチブル設計。バンジーレースで脱ぎ履きが速く、機内では踵を踏んで楽に履き、現地ではきちんと足を覆って歩ける。サンダルと靴を別々に持つ必要がない。

コンタクトの付け外し——アルコールティッシュ

長距離フライトでコンタクトを付けたまま寝るのは目への負担が大きい。途中で一度外してメガネに、着陸前に付け直す——という運用にしている人は多いはず。

その付け外しに欠かせないのがアルコールティッシュ。機内のトイレは混んでいて手を洗いに行くのも一苦労なので、座席で手指をその場で消毒できると安心だ。機内用と現地用に2パック持っていくのがおすすめで、食事前や化粧直しにも使えて出番が多い。

アルコール除菌ウエットティッシュ（100均で十分）

アルコール除菌 / メッシュタイプ / 個包装は100均で / まとめ買いはAmazon

機内のトイレが混んでいる時でも、座席で手指をその場で消毒できる。コンタクトを触る前に1枚出して拭くだけで、清潔さの不安が減る。食事前や化粧直しにも使えて出番が多い。旅行先でも普通に使うので、2パックほど持っていくと安心（機内用＋現地用）。機内に持ち込む分は100均の個包装タイプで十分で、ブランドにこだわる必要はない。家にストックしたい時だけ Amazon Basics などのまとめ買いパックから。

機内で使う小物は、小さなポーチにまとめる

有線イヤホン・2ピン変換・耳栓・アイマスク・アルコールティッシュ・リップクリーム・目薬・充電ケーブル・常備薬——機内で使うものは、小さなポーチにまとめてシートポケットに入れておくと出し入れが圧倒的に楽になる。仕切り付きのガジェットポーチで、シートポケットに入る幅・厚みが出ない薄型を選ぶのがコツ。

荷物の分け方——大型スーツケース＋機内持ち込みの2個体制

1週間ほどの海外滞在は、大型スーツケース1つ＋機内持ち込みキャリー1つの2個体制に落ち着いている。サムソナイトの大型とLOJELの機内持ち込みサイズの組み合わせ。

はじめは大型1つで足りると思っていたが、乗り継ぎ空港でターミナル間を歩く距離は思った以上に長い。お土産や上着・PCバッグを手で抱えて歩くと到着前に疲弊するので、機内持ち込みキャリーがあるだけで両手が空き、追加の荷物も転がして運べる。

もう1つ大きいのがロストバゲッジ対策。乗り継ぎが多い行程ほど預け荷物が遅延・紛失する確率は上がるので、1日分の着替え・常備薬・コンタクト・充電器は機内持ち込み側に必ず分けて入れる。これだけ守れば、万一届かなくても1〜2日は持ちこたえられる。

サムソナイト POLYGON SPINNER 75（109L）

Samsonite / 109L / 75cm / ハードシェル / 4輪スピナー / ※現在は廃盤

1週間の海外滞在で使っている預け入れ用の大型スーツケース。表面のポリゴン（多面体）デザインがサムソナイトらしく、ハードシェルで耐久性も高い。ただし現行モデルとしては廃盤のため、新規で買うなら同シリーズの後継や現行の大型ハードシェル（ESSENS／TRUNK／VOLANT 等）が候補になる。

まとめ

長距離フライトを楽にするコツは「全部揃える」ことではなく、自分のフライトスタイル（寝る派／映画派／作業派）に合わせて1〜2品ずつ試すこと。結果的に荷物も機内のストレスも減る。

個人的に外せないのは、有線イヤホン・2ピン変換・耳栓・アイマスク・アルコールティッシュ・小型ポーチ。スーツケースは大型＋機内持ち込みの2個体制が前提。あとはフライト時間や寝るかどうかで、空気式ピローと楽な靴を足すかを決めている。機内で「あれ持ってくればよかった」と後悔した記憶があれば、次のフライトでまず1つ試してみる、くらいで十分だ。

20時間超のフライトでも、ちょっとした準備の差で着いた瞬間の体力残量がだいぶ違う。詰め込まず、削りすぎず、自分の旅のスタイルに合うラインを探していくのが、いちばんの「快適」への近道だと思う。
